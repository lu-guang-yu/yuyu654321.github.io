最简单文件读写

  1. 发起文件读写请求：var fs = require(‘fs’);
  2. 读文件：fs.readFile(文件路径，回调函数);
    回调函数如：function（error, date);
    如果文件读取错误则error非空，date为定义。否则相反。
  3. 写文件：fs.writeFile(文件路径，内容，回调函数);
    回调函数如：function（error);
    如果文件读取错误则error非空。

最简单http服务

启动服务

  1. 发起服务请求：var http = require(‘http’);
  2. 开启服务器：var server = http.createServer();
  3. 响应处理函数：server.on(‘request’, 回调函数);
  4. 绑定端口号，启动服务器：server.listen(端口号如3000), 回调函数)

ctrl+c关闭服务

响应

request 请求事件处理函数有两个参数

  1. 请求对象Requset
  2. 响应对象Reponse

server.on(‘request’, function(request, reponse){})

request.url 客户路径 （端口号之后的路径）

reponse.write(响应内容)可有多个

response.end()结束响应
